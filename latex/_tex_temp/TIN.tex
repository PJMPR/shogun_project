\section{Cel dydaktyczny}

Zapoznanie studentów z nowoczesnymi technologiami internetowymi, które są kluczowe dla tworzenia i zarządzania dynamicznymi stronami oraz aplikacjami internetowymi. Zdobycie wiedzy na temat tworzenia responsywnych interfejsów, wykorzystania zaawansowanych technik CSS oraz programowania w języku JavaScript, w tym programowania funkcyjnego i obiektowego. Przedmiot obejmuje także tematykę asynchronicznego programowania, modyfikacji elementów DOM oraz korzystania z narzędzi sieciowych, takich jak HTTP i WebSocket. Dodatkowo, studenci poznają podstawy pracy z popularnym narzędziem do budowy aplikacji serwerowych.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{Introduction to modern web technologies that are essential for creating and managing dynamic websites and web applications. The course provides knowledge on building responsive interfaces, using advanced CSS techniques, and programming in JavaScript, including both functional and object-oriented approaches. It also covers asynchronous programming, DOM manipulation, and the use of networking tools such as HTTP and WebSocket. In addition, students learn the basics of working with a popular tool for building server-side applications.}

\section{Przedmiotowe efekty uczenia się}

\subsection*{Wiedza}
\begin{itemize}
  \item Student zna i rozumie podstawowe pojęcia protokołów internetowych: HTTP/HTTPS, REST, WebSocket, DNS, OAuth 2.0, CORS, zasady bezpieczeństwa webowego (XSS, CSRF, SQL injection) i architektury aplikacji internetowych (SPA, SSR, PWA). [K\_W09]
  \item Student zna i rozumie zaawansowane pojęcia z zakresu aplikacji internetowych: wzorce architektoniczne (MVC, MVP, MVVM, Clean Architecture), programowanie asynchroniczne (Promises, async/await) i nowoczesne frameworki frontend (React/Angular/Vue). [K\_W21]
\end{itemize}

\subsection*{Umiejętności}
\begin{itemize}
  \item Student potrafi ocenić przydatność różnych technologii webowych i wybrać odpowiednie środowisko do realizacji aplikacji internetowej zgodnie ze specyfikacją. [K\_U15]
  \item Student potrafi wytworzyć warstwową aplikację webową (frontend + backend + baza danych) z użyciem nowoczesnego stosu technologicznego, z uwzględnieniem bezpieczeństwa, wydajności i responsywności. [K\_U24]
  \item Student potrafi zdiagnozować problem specyficzny dla aplikacji webowej i zaprojektować jego rozwiązanie z doborem odpowiednich narzędzi i technologii. [K\_U28]
\end{itemize}

\subsection*{Kompetencje społeczne}
\begin{itemize}
  \item Student jest gotów do samodzielnego uczenia się nowych technologii webowych w dynamicznie zmieniającym się ekosystemie i regularnego aktualizowania swoich kompetencji. [K\_K03]
  \item Student jest gotów do współpracy w zespole nad aplikacją webową, przyjmując różne role i korzystając z narzędzi zarządzania projektem i wersjami kodu. [K\_K04]
\end{itemize}
