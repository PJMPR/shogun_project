\section{Cel dydaktyczny}

Celem przedmiotu jest zaznajomienie studentów z podstawowymi zagadnieniami uczenia maszynowego (w tym elementami uczenia głębokiego), głównie z wykorzystaniem sieci neuronowych (w tym sieci splotowych), a także z metodami programowania wybranych metod oraz praktycznym wykorzystaniem gotowych narzędzi.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to familiarize students with the basic issues of machine learning (including elements of deep learning), mainly using neural networks (including convolutional neural networks), as well as with methods for programming selected techniques and the practical use of ready-made tools.}

\section{Przedmiotowe efekty uczenia się}

\subsection*{Wiedza}
\begin{itemize}
  \item Student zna i rozumie zaawansowane pojęcia probabilistyki i statystyki: rozkłady prawdopodobieństwa, estymację, testy hipotez i ich zastosowanie w uczeniu maszynowym. [K\_W12]
  \item Student zna i rozumie zasady działania i zastosowania algorytmów uczenia maszynowego: regresja, klasyfikacja (SVM, drzewa decyzyjne, lasy losowe), klasteryzacja, redukcja wymiarowości (PCA) i metody ewaluacji modeli. [K\_W12]
  \item Student zna i rozumie zaawansowane pojęcia w zakresie sztucznej inteligencji: głębokie sieci neuronowe (CNN, RNN, Transformer), transfer learning i zasady odpowiedzialnej AI. [K\_W22]
\end{itemize}

\subsection*{Umiejętności}
\begin{itemize}
  \item Student potrafi zastosować aparat matematyczny (rachunek różniczkowy, algebrę liniową, statystykę) do implementacji i analizy algorytmów uczenia maszynowego w Pythonie (scikit-learn, PyTorch/TensorFlow). [K\_U07]
  \item Student potrafi zdiagnozować problem z obszaru sztucznej inteligencji, zaprojektować pipeline danych, wytrenować, ocenić i zoptymalizować model ML/DL na danych rzeczywistych. [K\_U29]
\end{itemize}

\subsection*{Kompetencje społeczne}
\begin{itemize}
  \item Student jest gotów do zastosowania sztucznej inteligencji na rzecz rozwiązywania problemów naukowych i społecznych z uwzględnieniem zasad etycznej AI. [K\_K01]
  \item Student jest gotów do samodzielnego śledzenia postępów w dziedzinie ML/AI i adaptowania nowych metod w praktyce inżynierskiej. [K\_K03]
\end{itemize}
