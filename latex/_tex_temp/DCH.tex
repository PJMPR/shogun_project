\section{Cel dydaktyczny}

Przedmiot obejmuje podstawy zarządzania bazami danych w środowiskach chmurowych. Studenci poznają architekturę usług bazodanowych dostępnych w nowoczesnych platformach chmurowych oraz uczą się korzystać z zarządzanych usług baz danych. W ramach zajęć omawiane są relacyjne i nierelacyjne systemy baz danych dostępne w modelu usługowym, w tym m.in. Amazon RDS, Amazon Aurora, Amazon Athena, a także magazyny danych typu klucz–wartość i pamięciowe systemy bazodanowe, takie jak Redis i DynamoDB.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The course covers the fundamentals of database management in cloud environments. Students learn about the architecture of database services available on modern cloud platforms and how to use managed database services. The classes discuss both relational and non-relational database systems available in the service model, including, among others, Amazon RDS, Amazon Aurora, Amazon Athena, as well as key–value data stores and in-memory database systems such as Redis and DynamoDB.}
