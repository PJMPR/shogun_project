\section{Cel dydaktyczny}

Celem przedmiotu jest zapoznanie się z techniką konteneryzacji na podstawie narzędzia Docker oraz Kubernetes. Efektem przedmiotu ma być możliwość utworzenia aplikacji typu REST API, która będzie spakowana za pomocą wyżej wymienionych narzędzi.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to familiarize students with containerization techniques using Docker and Kubernetes. The expected outcome of the course is the ability to create a REST API application packaged with the above-mentioned tools.}
