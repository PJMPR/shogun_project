\section{Cel dydaktyczny}

Celem przedmiotu jest zapoznanie studentów z podstawami tworzenia wieloplatformowych aplikacji mobilnych z wykorzystaniem frameworka Flutter i języka Dart. Studenci poznają architekturę aplikacji opartych na widżetach, zasady budowania interfejsu użytkownika, zarządzanie stanem oraz integrację aplikacji z zewnętrznymi usługami. Zajęcia mają charakter praktyczny i przygotowują do samodzielnego tworzenia prostych aplikacji działających na systemach Android i iOS.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to familiarize students with the basics of developing cross-platform mobile applications using the Flutter framework and the Dart language. Students learn the architecture of widget-based applications, the principles of user interface design, state management, and integration with external services. The classes are practical in nature and prepare students to independently create simple applications for Android and iOS.}
