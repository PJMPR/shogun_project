\section{Cel dydaktyczny}

Celem kształcenia jest nabycie umiejętności tworzenia obrazów z wykorzystaniem standardowego API graficznego (biblioteki OpenGL i renderera Blender) oraz realizacji podstawowych transformacji obrazów 2- i 3-wymiarowych.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to acquire the ability to create images using standard graphics APIs (the OpenGL library and the Blender renderer) and to perform basic 2D and 3D image transformations.}
