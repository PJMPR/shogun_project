\section{Cel dydaktyczny}

Celem przedmiotu jest zapoznanie studentów z podstawami programowania funkcyjnego w języku F\# oraz wykształcenie umiejętności tworzenia prostych aplikacji działających na platformie .NET. Zajęcia kładą nacisk na niemutowalność danych, funkcje wyższego rzędu, dopasowanie wzorców oraz bezpieczne modelowanie danych. Przedmiot przygotowuje studentów do pracy z nowoczesnymi paradygmatami programowania oraz do świadomego wyboru narzędzi w środowisku .NET.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to familiarize students with the basics of functional programming in F\# and to develop the ability to create simple applications running on the .NET platform. The course places emphasis on data immutability, higher-order functions, pattern matching, and safe data modeling. It prepares students to work with modern programming paradigms and to make informed tool choices within the .NET environment.}
