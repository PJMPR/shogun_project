\section{Cel dydaktyczny}

Celem przedmiotu jest przekazanie wiedzy i rozwinięcie umiejętności z zakresu administracji systemami IT w środowisku korporacyjnym, w szczególności instalacji, konfiguracji i zarządzania systemem Windows Server oraz usługami sieciowymi, z uwzględnieniem aspektów bezpieczeństwa i automatyzacji.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to provide knowledge and develop skills in IT systems administration in a corporate environment, in particular the installation, configuration, and management of Windows Server and network services, with consideration of security and automation aspects.}

\section{Przedmiotowe efekty uczenia się}

\subsection*{Wiedza}
\begin{itemize}
  \item Student posiada wiedzę dotyczącą architektury systemu Windows Server, jego ról i usług oraz zasad instalacji, konfiguracji i aktualizacji systemu serwerowego.
  \item Rozumie koncepcję domeny, kontrolera domeny, usług Active Directory, DNS i DHCP oraz mechanizmów zarządzania użytkownikami, grupami i uprawnieniami.
  \item Zna podstawy bezpieczeństwa systemów serwerowych, polityk zabezpieczeń, kopii zapasowych i odtwarzania danych.
  \item Rozumie zasady wirtualizacji, monitorowania wydajności oraz automatyzacji.
\end{itemize}

\subsection*{Umiejętności}
\begin{itemize}
  \item Student potrafi instalować i konfigurować system Windows Server w środowisku fizycznym i wirtualnym.
  \item Konfiguruje domenę oraz usługi katalogowe i sieciowe, zarządza kontami użytkowników, grupami i politykami GPO.
  \item Wdraża usługi plikowe i sieciowe, monitoruje działanie systemu oraz diagnozuje i usuwa typowe problemy administracyjne.
  \item Wykonuje kopie zapasowe, odtwarza dane oraz automatyzuje zadania administracyjne.
  \item Stosuje mechanizmy zabezpieczeń w celu ochrony danych i infrastruktury.
\end{itemize}

\subsection*{Kompetencje społeczne}
\begin{itemize}
  \item Student jest gotowy do samodzielnej i zespołowej pracy przy utrzymaniu infrastruktury serwerowej.
  \item Przejawia odpowiedzialność za bezpieczeństwo danych, ciągłość działania systemów i poprawność konfiguracji.
\end{itemize}
