\section{Cel dydaktyczny}

Celem przedmiotu jest zapoznanie studentów z podstawami programowania w języku C++ oraz wykształcenie umiejętności tworzenia prostych aplikacji konsolowych z wykorzystaniem klasycznych konstrukcji programistycznych. Zajęcia kładą nacisk na zrozumienie modelu pamięci, poprawne użycie typów danych, funkcji i struktur, a także wprowadzenie do podstaw programowania obiektowego. Przedmiot przygotowuje studentów do dalszej nauki języków kompilowanych oraz pracy nad projektami wymagającymi większej kontroli nad wydajnością i zasobami.

\subsection*{\textit{Wersja w języku angielskim (English version)}}

\textit{The aim of the course is to familiarize students with the basics of programming in C++ and to develop the ability to create simple console applications using classical programming constructs. The course places emphasis on understanding the memory model, correct use of data types, functions and structures, as well as an introduction to the basics of object-oriented programming. It prepares students for further study of compiled languages and for work on projects requiring greater control over performance and resources.}
